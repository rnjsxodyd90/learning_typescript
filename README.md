# 7-Day Interview Prep: Angular + TypeScript + Java Spring Boot

## Your Goal
Learn the basics of Angular, TypeScript, HTML/CSS, and Java Spring Boot for a junior software engineer interview.

## How To Use This
- Go day by day, in order
- READ every file carefully — don't skim
- TYPE the code yourself — don't copy paste
- After each file, try the exercises
- If you don't understand something, re-read it or Google it

## Schedule

| Day | Folder | Topic | Time |
|-----|--------|-------|------|
| 1 | `day1-typescript/` | TypeScript fundamentals | 3-4 hours |
| 2 | `day2-html-css/` | HTML and CSS basics | 3-4 hours |
| 3 | `day3-angular-basics/` | Angular components & templates | 4-5 hours |
| 4 | `day4-angular-services/` | Angular services, HTTP, routing | 4-5 hours |
| 5 | `day5-java-basics/` | Java language fundamentals | 4-5 hours |
| 6 | `day6-spring-boot/` | Spring Boot REST API | 4-5 hours |
| 7 | `day7-review/` | Interview questions & cheat sheets | 3-4 hours |

## Setup You Need
- Node.js (you already have this)
- Angular CLI: `npm install -g @angular/cli`
- Java JDK 17+: Download from https://adoptium.net/
- An IDE: VS Code (for Angular) + IntelliJ IDEA Community (for Java)

## Key Interview Concepts You MUST Know
1. What is TypeScript and why use it over JavaScript?
2. What is a component in Angular?
3. What is dependency injection?
4. What is a REST API?
5. What are HTTP methods (GET, POST, PUT, DELETE)?
6. What is Spring Boot?
7. What is the difference between frontend and backend?

Lesson 3: CSS Basics

CSS makes HTML look good. Let's learn how it works.

CSS Selectors: How to TARGET Elements

A selector tells CSS WHICH elements to style. There are 3 main types:

1. Element Selector (targets ALL of that tag type)

This <h1> at the top is purple because of: h1 { color: #9C27B0; }

Every single <h1> on this page will be purple. You can't make just ONE purple with an element selector.

2. Class Selector (targets elements with a specific class -- MOST USED)

This text has a highlighted class applied to it.

This text is green and bold because of the .success-text class.

This text is red and bold because of the .error-text class.

3. ID Selector (targets ONE specific element)

This paragraph has id="special-paragraph" so it gets unique styling. There should only be ONE element with this ID on the entire page.

Summary:
p { } = element selector, targets ALL <p> tags
.classname { } = class selector, targets elements with that class (DOT)
#idname { } = ID selector, targets ONE element with that ID (HASH)

In practice: Use CLASSES 90% of the time. They're the standard.

There are many ways to write colors in CSS:

red
#FF5722
rgb()
rgba()

color: red; (text color by name)

color: #2196F3; (text color by hex code -- this is blue)

background-color: #FFEB3B; (yellow background)

Tip: Google "color picker" to find hex codes for any color. You do NOT need to memorize hex codes. Nobody does!

Font Properties

Font Size:

font-size: 12px; -- small text

font-size: 18px; -- medium text

font-size: 28px; -- large text

Font Family (the typeface):

font-family: Georgia, serif; -- Serif (has little feet on letters)

font-family: Arial, sans-serif; -- Sans-serif (clean, no feet)

font-family: "Courier New", monospace; -- Monospace (for code)

Font Weight, Style, and Decoration:

font-weight: bold; -- Bold text

font-weight: 300; -- Light/thin text

font-style: italic; -- Italic text

text-transform: uppercase; -- uppercase text

text-decoration: underline; -- Underlined text

The Box Model (VERY IMPORTANT)

Every HTML element is a BOX with 4 layers. Think of it like this:

MARGIN (the yard / empty space outside)
BORDER (the fence)
PADDING (your jacket / cushioning)
CONTENT (you! the text/image)

Here's a box with all 4 layers visible:

I am the content!
width: 300px
padding: 20px (jacket)
border: 3px solid (fence)
margin: 30px auto (yard)

Padding examples (space INSIDE the border):

padding: 5px; (very tight)
padding: 20px; (comfortable)
padding: 40px; (very spacious)

Margin examples (space OUTSIDE the border):

margin: 5px; (close to neighbors)
margin: 30px; (far from neighbors)

Margin and Padding shorthand:

padding: 10px; -- same on ALL sides (10px top, right, bottom, left)
padding: 10px 20px; -- 10px top/bottom, 20px left/right
padding: 10px 20px 30px 40px; -- top, right, bottom, left (clockwise!)
padding-top: 10px; -- only the top

Same rules apply for margin!

Borders

The border property has 3 parts: width, style, and color.

border: 1px solid black; (thin, solid line)
border: 3px dashed #2196F3; (thicker, dashed, blue)
border: 5px dotted #f44336; (thick, dotted, red)
border-radius: 10px; (rounded corners!)
border-radius: 50%; (perfect circle!)
border-radius rounds the corners. The bigger the number, the more rounded. 50% makes a perfect circle (if width and height are equal). This is super commonly used for profile picture avatars!

Width and Height

width: 200px
height: 80px
width: 50% (half of parent)
width: 100% (full width of parent)
px vs %:
px (pixels) = fixed size. 200px is always 200px.
% (percent) = relative to the parent element. 50% = half of the parent's width.

Pro tip: Add box-sizing: border-box; to make width/height INCLUDE padding and border. Without it, padding and border are ADDED to the width, which causes elements to be bigger than you expect. Most modern projects set this globally.
